A posteriori error analysis for Diatomic 1-D molecules

1D toy problem:
---------------
with RHS (or eigenvalue problem)
(- (1/2) \Delta + V + \lambda) u = f 
for V user-defined potential

Solvers:
--------
runge-kutta, galerkin, finite difference

Supported galerkin basis:
-------------------------
gaussian, Hermite-gaussian

Other:
------
Quadrature for Slater orbitals
